# 作業 3 基礎 PHP 語法

## 說明
本作業共有 4 題，請分別完成於 p01.php p02.php p03.php p04.php

### p01.php
隨機產生 0 - 1000 的 100 個數值，存放在一個陣列中。撰寫一個檢查一個數值是否為質數的函式。接著使用這個函式檢查陣列中的數值是否為質數。設計一個表格，將這些數值列出，標示是否為質數。

### p02.php
使用讀取時間日期的內建函式，在畫面上列印出 年月日 時分秒 。假設分別為 y, m, d, h, n, s 。請計算這幾個數值相加的總和。另外檢查這個數值是否為質數。

### p03.php 
根據所給的 data.csv 資料，撰寫程式讀取這個檔案的內容，請計算各項總和並且將資料以表格表示。

### p04.php
根據所給的 graph.csv 資料，撰寫繪圖程式將這些數據以視覺化表示。

1. 長條圖：X 軸放商品分類，Y 軸放營業額或銷售件數
2. 圓餅圖：用各商品分類的營業額占比
